## User-story: ...

**As a** Mechanic.
**I want** to able to keep track of the services that are being done for the customer, and also to be able to share the knowledge that the mechanic has to other mechanics that are struggling or just starting.
**So that** I can organize which services I have completed and so I can order parts that are out of stock, I want to be able to help other mechanics with the knowledge that i have.

## Acceptance Criteria:
    -  Implementing a task tracker that helps the mechanic to tick off which jobs they have completed that helps them complete their work more effectively and organized.
    - A live chat for sharing photos, videos and text among each other, for fast replies when they are stuck with an issue.
    - An ordering system that notifies the stockers to order new stock when the mechanic lets them know there are no more parts left.
